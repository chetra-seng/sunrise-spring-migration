# Sunrise Spring Migration

Slidev presentation for the Sunrise Java Course covering API documentation with Swagger/OpenAPI and database schema management with Liquibase.

## Topic

**Documentation, Migration & Review** — making the Task Flow API production-ready with:
- Swagger/OpenAPI for API documentation
- Liquibase for database migrations

## Usage

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Export to PDF
pnpm export
```
